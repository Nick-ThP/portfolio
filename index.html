Welcome


My name is Nicklas Poulsen and I am an educated UI/UX Designer and Information Architect (Msc IT). After receiving my master's degree in the summer of 2021, I decided to examine the fundamentals of JavaScript to support my existing knowledge of HTML/CSS and get a better understanding of web development practices.

Originally this approach was meant as a way to become better at communicating with developers. However, I started to see a natural connection between my existing design/architecture knowledge and application development with JavaScript frameworks like React. I then decided that I wanted to make my career more development-oriented.

Motivation


I am motivated to design and build digital products that users find both interesting and intuitive. Something that users keep coming back to again and again without even thinking about it. What initially pushed me towards web design was the amount of unfriendly and bloated web experiences. I wanted to do something about that.


I have worked full-time for about one year and am enjoying the experience. I am starting to get a good grasp on frontend problem solving. I also really enjoy the social aspect of working with other people. Try clicking the circle in the middle. Can you make the interface disappear completely for at least 3 seconds with carefully placed clicks?

Skillset


The toolbox of practical theories and methodology that I have acquired throughout my studies has a broad scope in terms of applicability. Subjects such as design psychology, audience analysis, media sociology, test strategies, data handling, wireframing etc. all ties into my overall skillset and authenticity in terms of participating in design and architecture.


My core strength as a developer is my design perspective and ability to analyse and evaluate the interface so that it serves the user properly. There is a lot of ways that this knowledge comes into play when developing applications in React. Aspects such as interactivity, loading procedures and error handling are all pieces of a succesful user experience. Besides that I am also a sharp programmer.

I started out with the basics of JavaScript and React through Codecademy's Frontend Career Path. It helped me get started with a structured and up-to-date curriculum. I have also utilized platforms like Udemy to dive into individual web technologies such as Redux, Next.js, Material UI, Sass and Firebase + many more. At this point I have also worked full-time in a frontend position for about a year.

Simple Weather


This is a personally designed and fully typed weather application which utilizes the OpenWeatherMap API to showcase data. The app doesn't rely on any CSS frameworks since I wanted to challenge my perception of vanilla CSS and responsive design. The app, although simple in nature, has been a great way for me to affirm and perpetuate some of my core knowledge of not just React but also UI/UX design principles.







Movie Database


This is a fetching application made with React. It fetches and displays data about movies and tv shows from a publicly accessible API. There is filtering, pagination and the possibility to index favorite movies in client cache as well. The project showcases fundamental React understanding with hooks such as useState and useEffect. It also shows dictation of rendered content based on ternary operators.







Expense Tracker


This is a React budgeting tool to help create an overview over personal finances. It is made with Material UI and Chart.js in order to deliver a good user experience. State is managed by utilizing the useContext hook. There is also a custom React hook present for managing transactions. Local storage makes it possible to store and render previous transactions without much hassle.







Note Keeper


This is a very basic and minimal to-do application made without any libraries. It is lightweight and reliable since it does not require a lot of code to deliver the required user experience. It works with local storage so that the user can always easily access his/her notes. This streamlines the experience and makes it the ideal app to pull up when doing everyday tasks like grocery shopping.
